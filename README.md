# devGPT AI – AI Chatbot for Developers

Your personal AI coding assistant that helps to debug, writes code, explains concepts — instantly!

Live Demo: https://devgptai.vercel.app

### Features

- Chat with powerful AI (Grok / Claude / GPT-4o)
- Code generation, debugging, explanations
- Beautiful **Dark & Light Mode**
- Sign in with **Email** or **Google** (one-click)
- Free daily messages + **Khalti Payment** to buy more credits (Nepal)
- Khalti Test Mode included (no real money needed during dev)
- Responsive design – works on mobile & desktop

### Tech Stack

- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB
- **Auth**: JWT + Google OAuth + HttpOnly cookies + localStorage fallback
- **Payment**: Khalti Payment Gateway (Nepal)
- **Deployed**: Vercel (frontend + backend serverless)
